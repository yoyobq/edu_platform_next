# Logo

Logo 表达"这是哪个产品"。不表达"这里能点"或"这是一个状态"。

## 当前状态

| 项目                   | 现状                                                                   |
| ---------------------- | ---------------------------------------------------------------------- |
| Logo 图形              | 字母 "ai" 与对话气泡融合的抽象标识（C 形弧线 + 气泡 + i 竖杆 + 红点）  |
| 顶部品牌位             | 文字品牌（`Typography.Title level={4}` 渲染产品名），未使用图形 Logo   |
| 公共入口页左栏         | `Typography.Text secondary` 渲染产品名（小字号品牌标识）+ 页面标题     |
| `public/logo.svg`      | 彩色版位图包裹型 SVG（512×512 内嵌 PNG）；可暂用于对外截图等低要求场景 |
| `public/logo-dark.svg` | 深色背景版占位文件；当前为 `logo.svg` 副本，需真矢量替换               |

后续若引入图形 Logo 到页面渲染，需提供真矢量 SVG 并替换对应 `public/` 文件。

## 允许位置

| 位置            | 说明                                |
| --------------- | ----------------------------------- |
| 顶部品牌位      | "Logo + 产品名"或纯文字品牌，二选一 |
| 公共入口页左栏  | 品牌展示区；不放进右侧操作 Card     |
| 对外截图 / 封面 | 品牌归属场景                        |

## 禁止位置

- 正文卡片头部重复放 Logo
- 列表项、表格行、Tag、按钮内部
- 充当 AI 功能图标或操作 icon
- 同一屏重复出现多个同级 Logo

## 尺寸

| 场景         | 推荐高度 | 下限   |
| ------------ | -------- | ------ |
| 顶部品牌位   | `24px`   | `20px` |
| 公共入口左栏 | `32px`   | `20px` |

## 间距与对齐

- 四周保留独立留白，不贴边框、Tag、按钮或分割线
- 与文字品牌并排：`gap-2`
- 优先结构对齐；图形重心偏移时允许极小幅度视觉微调（仅修正平衡）

## 深色模式

| 背景     | 使用版本       |
| -------- | -------------- |
| 浅色     | 标准品牌配色   |
| `.dark`  | 反白版或深底版 |
| 无专版时 | 回退为文字品牌 |

禁止改成功能态颜色（成功绿、告警黄、AI 青色）。

## 弱露出

- 顶部品牌位已存在 → 正文区域不追加第二次品牌露出
- 确需弱露出 → 背景级、低对比、不可交互（如页脚弱水印）
- 不得影响正文可读性、表单可用性、表格扫描
- 让页面更重或更吵时不做

## 禁止变形

禁止拉伸、压扁、旋转、加描边、加阴影特效、拆开元素重组、放进按钮充当 icon。
