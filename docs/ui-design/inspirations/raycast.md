# Raycast (AI 交互与快捷面板标杆)

## 风格关键词

高密度、键盘优先、丝滑流畅、打字机流式输出、工具感。

## 为什么选择 Raycast？

作为一个带有 AI 属性的教育平台，我们需要一个能在“高密度快捷操作”与“AI 丝滑生成”之间做到完美平衡的参考。Raycast 虽然是 macOS 桌面端应用，但其对**键盘交互的极致追求**和**AI 内容的流式展现**，是全行业公认的顶级水准。

⚠️ **警告：本项目禁止复刻 Raycast 的“黑曜石冷峻风格”视觉皮囊。** 我们是面向师生的平台（专业、亲切），不是纯极客开发者工具。本篇文档**仅用于提取交互与排版模式**，所有视觉落地必须使用项目中现有的 Tailwind Token 和 Ant Design 主题。

---

## 核心可吸收点（交互与排版）

### 1. 全局命令面板（Command Palette / Cmd+K）

如果我们在项目中实现全局搜索或快捷命令面板，应当以 Raycast 为最高标准：

- **键盘优先的导航**：用户完全不需要鼠标。上下键切换列表项时，焦点状态（Focus/Selected）的切换必须丝滑、没有延迟。
- **高密度列表排版**：在极小的空间内，合理安排图标、主标题、次标题（Subtitle）和快捷键提示。
- **视觉层级**：通过文字颜色的明暗（如：主标题用高亮色，次标题和快捷键用灰色）来拉开层级，而不是依赖字号大小的剧烈变化。

### 2. AI 助手对话区（antdX Sidecar 适用）

- **流式输出（Streaming）体验**：AI 回复时，打字机效果的节奏感。
- **加载骨架（Skeleton）**：在等待 AI 响应的短暂时间里，使用精致的骨架屏（结合 `duration-200` 过渡）代替生硬的 Spinner。
- **代码块与结构化数据**：AI 输出代码块、表格时，边距控制得当，且具备一键复制/执行的快捷入口。

### 3. 快捷键视觉（`<kbd>` 元素）

- 对快捷键的精致刻画。我们可以利用项目自身的 token（如轻量的背景色 + subtle border + 小圆角 `--radius-badge`）来实现干净、现代的按键提示，帮助高级用户（如老师）提升效率。

## 🚫 不可照搬点（严格禁止的视觉特征）

以下是 Raycast 营销页和原生应用中的典型特征，**绝对禁止**出现在我们的业务代码中：

1. **极度深色的冷色调背景**：禁止使用类似 `#07080a` 的深蓝黑色调。我们的暗色模式（Dark Mode）需遵循项目自身的配色体系，保持适度的温度。
2. **macOS 原生的多层内发光阴影**：禁止手写 `inset rgba(...)` 这种复杂阴影来模拟物理厚度。阴影必须消费 `tokens.md` 中规定的 `--ant-box-shadow-secondary` 等标准 Token。
3. **强烈的红色点缀（Raycast Red）**：我们的品牌色和 AI 强调色（Claude Coral）已有明确规定，禁止引入外部品牌色。
4. **强行修改字体排版规则**：禁止为了模仿其“空气感”而全局增加字间距（如 `letter-spacing: 0.2px`），也禁止脱离 Ant Design 默认字号去重新定义一套排版比例。
5. **特殊的组件形态（如 86px 药丸按钮）**：组件形态（如圆角、边框）必须尊重 Ant Design 的组件默认设定，禁止通过强行覆盖 CSS 破坏组件原本的结构。

## 在本项目中的适用范围

- **高度适用**：
  - AI Sidecar（侧边栏）对话流、输入框交互反馈。
  - 全局搜索弹窗、命令面板（如果有）。
  - 下拉菜单（Dropdown Menu）中复杂的层级和快捷键展示。
- **不适用**：
  - 常规的数据表格、表单页、Dashboard 面板。
  - 公共入口页（Login/Signup）。

## 给 AI 与开发者的落地要求

当你被要求“参考 Raycast 的交互”时，你的代码实现应当是：

1. **结构上**：借鉴其紧凑、高密度的 DOM 嵌套和 Flex/Grid 布局方式。
2. **样式上**：**必须**将其视觉语言翻译为我们自己的语言。
   - 比如：它的“高亮蓝色选中态”，你需要翻译为 `bg-fill-hover` 或主题色 Token。
   - 比如：它的“精密阴影浮层”，你需要翻译为 `shadow-surface` 或 `shadow-card`。
   - 比如：它的快捷键提示，你需要使用 `text-xs text-text-tertiary bg-fill-quaternary rounded-badge` 来实现。
